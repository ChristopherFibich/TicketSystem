Help

Basics
  • {{ user.username }} is shown in the top bar.
  • Use {{ user.username }}'s Tickets to view what is assigned to you.
  • Tap a ticket to edit it, change status, or mark it done.
Ticket status and scoring
  • Mark done (+points) creates a completion entry and updates the scoreboard.
  • If a ticket/template is set to Counts for score = off, it awards 0 points (shown as “unscored”).
  • The scoreboard uses points (not number of tasks).
Ticket colors
Blue means done; otherwise colors indicate how long it has been assigned.
  • Blue: done
  • Green: assigned for < 1 day
  • Yellow: assigned for 1–5 days
  • Red: assigned for 5+ days
Scoreboard
  • Bar charts show points earned: today, last 7 days, last 30 days.
  • Use Details to expand a person’s completed ticket titles and points.

Admin / recurring templates (superusers)
  • Go to /admin/ to create users and recurring templates.
  • Create templates under Ticket templates (daily/weekly/monthly).
  • Assignment mode:
    • Fixed: always assigned to a specific person.
    • Eligible pool: assigned fairly based on lifetime points.
  • After creating templates, run the spawner to generate tickets:
    • python manage.py spawn_recurring_tickets
  • All tickets shows every ticket and who it belongs to.
